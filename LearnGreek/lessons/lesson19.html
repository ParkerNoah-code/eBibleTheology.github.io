Lesson 19: Perfect Participles

1. Perfect Active Participles (Masculine)

Formation

  • Reduplication (e.g., λελυ-)
  • Addition of kappa (κ) (e.g., λελυκ-)
  • No connecting vowel

Paradigm Example (λυω)

Case Singular Plural
Nominative λελυκώς λελυκότες
Genitive λελυκότος λελυκότων
Dative λελυκότι λελυκόσι(ν)
Accusative λελυκότα λελυκότας

Follows third declension endings.

2. Perfect Middle/Passive Participles (Masculine)

Formation

  • Reduplication
  • Infix -ομεν-
  • Adjectival ending

Paradigm Example (λυω)

Case Singular Plural
Nominative λελυμένος λελυμένοι
Genitive λελυμένου λελυμένων
Dative λελυμένῳ λελυμένοις
Accusative λελυμένον λελυμένους

Follows second declension endings.

3. 2nd Perfect Participles (No κ)

  • ἀκούω → ἀκηκοώς
  • ἀνοίγω → ἀνεῳγώς
  • γίνομαι → γεγονώς
  • ἔρχομαι → ἐληλυθώς
  • λαμβάνω → εἰληφώς
  • πείθω → πεποιθώς

4. Usage of Perfect Participles

  • Emphasizes completed action or resulting state
  • Functions:
    • Attributive: modifies a noun (e.g., τὸν ἐσπαρμένον)
    • Substantive: stands alone (e.g., ὁ γεγεννημένος)
    • Adverbial: describes state (e.g., ἀνεῳγότα)

5. Genitive Absolute

  • Adverbial participle + subject in genitive case
  • Loosely connected to main clause
  • Translation:
    • Present participle → “while”
    • Aorist participle → “after”

Example: αὐτοῦ λαλοῦντος → “as he was saying”

6. Periphrastic Participles

  • Verb εἰμί + participle
  • Common in Luke’s writings
  • Example: ἦν διδάσκων → “he was teaching”

7. Future Participles

  • Active: stem + σ + οντος (e.g., λύσοντος)
  • Middle: stem + σ + ομένου (e.g., λυσομένου)
  • Passive: stem + θησ + ομένου (e.g., λυθησομένου)
  • Only 13 in NT; expresses purpose
